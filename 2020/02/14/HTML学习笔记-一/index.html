HTML学习笔记(一)
发表于 2020-02-14| 更新于 2020-02-16|HTML学习

cover: pid=79453950

在开始学习HTML之前，先简单了解几个关于HTML的基本概念。

什么是HTML？

  • HTML（Hyper Text Markup Language) 即超文本标记语言
  • HTML不是一种编程语言，而是一种标记语言。
  • 标记语言指的是一套标记标签
  • HTML使用标记标签来描述网页

什么是HTML标签？

  • HTML标签是由尖括号包括的关键词，例如<html>
  • HTML标签一般是成对出现的，如<b>、</b>
  • 带”/“的是结束标签，不带的是开始标签
  • 开始和结束标签有时被称为开放标签和闭合标签

什么是HTML文档？

  • HTML文档即网页，是用来描述网页的

  • HTML文档包含HTML标签和纯文本

  • Web浏览器读取HTML文档，并使用HTML标签来解释网页内容，例如:

    html
