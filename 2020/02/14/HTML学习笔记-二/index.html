HTML元素

什么是HTML元素

  • HTML元素指的是从开始标签到结束标签的所有代码。

    开始标签 元素内容 结束标签
    <p> Paragragh. </p>
    <a href=”default.htm”> link </a>
    <br/>

HTML元素语法

  • HTML元素以开始标签起始，以结束标签终止。
  • 元素的内容是开始与结束标签之间的内容。
  • HTML元素可以具有空内容

嵌套的HTML元素

  • HTML文档大多由嵌套的元素构成。

    实例

    html

    在这个简单的HTML文档实例中，我们不妨稍微分析一下其中的各个元素。

    • <p>元素：

      html

      一对<p>、</p>标签包含一个段落元素，“Hello，world!”是元素内容，该元素定义了HTML文档中的一个段落。

    • <body>元素：

      html

      <body>元素定义的是HTML文档的主体，在这里它包含的是另一个HTML元素（p元素）。

    • <html>元素：

      html

      <html>元素定义的是整个HTML文档的边界，在这里元素的内容是另一个HTML元素（body元素）。

空元素

没有内容的元素称为空元素，空元素在开始标签中同时关闭，即可以没有关闭标签，如<br>标签定义换行。

关闭空元素可以用<br/>,虽然<br>在大多数浏览器中有效，但是使用<br/>是更规范且长远的用法。

关于HTML的大小写敏感

HTML标签对大小写不敏感。
但万维网联盟（W3C）在 HTML 4 中推荐使用小写，而在未来 (X)HTML 版本中强制使用小写。

HTML属性

什么是HTML属性

  • 属性为HTML元素提供附加信息
  • 属性总是以名称/值对的形式出现，如：name=”value”
  • 属性总是在HTML元素的开始标签中规定

属性值记得加引号

  • 属性的值必须包括在引号内，使用双引号或单引号都没问题，但在某些情况下属性值本身就含有双引号，此时必须使用单引号。
