HTML标题

标题(Heading)通过<h1>-<h6>定义。
<h1>定义最大的标题，<h6>定义最小的标题。
eg.

html
  • 浏览器自动在标题前后添加空行。
  • 默认情况下，浏览器自动在块级元素前后都添加一个额外的空行，比如段落、标题元素。

为什么仅能将<h>标签用于标题？

  • 搜索引擎使用标题为网页的结构和内容编制索引，标题用于呈现文档结构是很重要的。

HTML换行

在不产生一个新段落的情况下进行换行，使用<br />标签。
eg.

html
  • <br />没有结束标签，而且省略“/”大多数情况下没有问题，但是请尽量使用<br />，在XHTML、XML和将来的HTML中不允许使用未闭合的标签。
  • 通过在<br />中添加clear属性（left、right、all）指定边界，可以使文本流在换行后指定的边界上没有图像时，才会继续输出文本。用以实现在图像或表格的下方继续输出文本。

HTML中的空格

  • 浏览器在显示页面时，会忽略源代码中的多余的空格和空行，但并不是移除所有空格和空行，所有连续的空格或空行都会被显示为一个空格。
  • 每行容纳的字数取决于浏览器窗口大小。

HTML样式

HTML的style属性

  • style属性的作用:

    提供了一种改变所有HTML元素的样式的通用方法。

    样式从HTML 4 开始引入，他是一种新的首选的 改变元素样式的方法。通过 HTML 样式，能够通过 style 属性直接将样式添加到元素中，或者间接地在独立的样式表中（CSS 文件）进行定义。

不赞成使用的一些标签和属性

在 HTML 4 中，有若干的标签和属性是被废弃的。被废弃（Deprecated）的意思是在未来版本的 HTML 和 XHTML 中将不支持这些标签和属性

  • 避免使用以下这些被废弃的标签和属性！

    标签 描述
    <center> 定义居中的内容
    <font>和<basefont> 定义HTML字体
    <s>和<strike> 定义删除线文本
    <u> 定义下划线文本
    属性 描述
    bgcolor 定义背景颜色
    align 定义文本的对齐方式
    color 定义文本颜色

    以上这些标签属性，都可以用style代替。
