HTML引用

短引用<q>

  • 定义短引用
  • 浏览器通常会为<q>元素包围引号
html

长引用<blockquote>（也叫块引用）

  • 长引用定义一个单独的描述引用内容的节
  • 浏览器对长引用元素进行缩进，并且上下自动产生空行。
html

缩略词<abbr>

  • 定义元素缩写或首字母缩略语
  • 对缩写进行标记能够为浏览器、翻译系统以及搜索引擎提供有用的信息。
html
  • 添加tltle属性描述缩写内容

定义<dfn>

  • 定义项目或缩写,一般斜体 显示。

  1. 如果设置了 <dfn> 元素的 title 属性，则定义项目：

    html

  2. 如果<dfn> 元素包含具有标题的 <abbr> 元素，则 title 定义项目：

    html

  3. 否则，<dfn> 文本内容即是项目，并且父元素包含定义。

    html
  • 简而化之可以用<abbr>代替，或者用第一条。

地址<address>

  • 定义文档或文章的联系信息，通常以斜体 显示

    html

著作标题<cite>

  • 定义著作的标题

  • 通常以斜体显示

    html

双向重写<bdo>

  • 定义双流向覆盖

  • 用于覆盖当前文本方向, 用dir属性指定
